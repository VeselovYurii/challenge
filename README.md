# challenge

Prerequisites
-------------

* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/)
* [npm](https://www.npmjs.com/)

Installation
------------

Clone the project:

```bash
git clone https://github.com/VeselovYurii/challenge.git
```

Install the dependencies:
------------


cd challenge
cd server


in both derictories:
------------

```bash
npm install
```


Start the client
------------

```bash
npm run start
```

Start the server:
------------

```bash
npm start serve
```
